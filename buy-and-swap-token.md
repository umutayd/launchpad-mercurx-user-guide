# 🤑 Buy & Swap Token

Click on projects to make purchases and swap transactions on active projects ;

1. Token value, time remaining, total collected are shown on the card in these screens.
2. There are 3 different situations: add, buy and swap.
3. In the lower section of these screens, there are sales information, token information, information about the project.
