# 💸 Sales

All sales are listed on [this screen](https://launchpad.mercurx.com/sales);&#x20;

1. A search bar is available to search by project name, token symbol, and token contract address.
2. It is possible to sort by token name, token symbol and token address.
3. There are buttons for listing in 3 different types: all, active and completed.
